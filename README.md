# webserver

## install pre-requisites
```$ sudo apt-get install -y nodejs```

## start server
```$ node app.js```

## open browser to site
```$ google-chrome http://localhost:1337```


# express-app

## install pre-requisites
```$ sudo apt-get install -y nodejs npm
$ npm install express
$ npm install mongoose
$ npm install ejs```

## start server                 
```$ node app.js```

## open browser to site
```$ google-chrome http://localhost:1337```
